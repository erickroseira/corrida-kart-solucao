Para que a solução funcione corretamente é necessário assegurar que os seguintes tópicos serão atendidos:

-> 1. A solução foi desenvolvida em Java. Neste caso é mandatório versão Java 8, pois a solução utiliza recursos que somente estão 
      disponíveis a partir do Java 8. Nesta solução foram utilizadas as API Streams e funções Lambda, recursos desta versão do Java.
      
-> 2. Baseado-se no formato do log disponibilizado é necessário que os dados das colunas sejam separados por mais de um espaço. Ou seja,
      o código irá definir o valor das colunas por dois caracters de espaço ou mais (A definição do valor de cada coluna se dá pela
      identificação de 2 caracteres de espaço ou mais). Por exemplo, o log abaixo está no formato correto, pois o valores de cada coluna
      estão separados por apenas 1 espaço. Na coluna Piloto por exemplo o código, o caractere - e o nome do piloto (038 – F.MASSA) estão
      separados por 1 espaço. Já valores de colunas diferentes estão separadas entre si, por mais de um  espaço.
      
      Hora                               Piloto             Nº Volta   Tempo Volta       Velocidade média da volta
      23:49:08.277      038 – F.MASSA                           1		1:02.852                        44,275
      
      Por outro lado o exemplo, abaixo irá gerar execução e dados incorretos. Pois o codigo e o nome do Piloto está separado por mais 
      de um espaço. Desta forma o programa irá entender que tem 7 colunas, quando na realidade temos 5.
      
      Hora                               Piloto             Nº Volta   Tempo Volta       Velocidade média da volta
      23:49:08.277      038    –    F.MASSA                           1		1:02.852                        44,275
      
      ENTÃO ASSEGURE DE ESTAR SEGUNIDO ESTE PADRÃO NO ARQUIVO DE LOG
     
 -> 3. O projeto java foi criado no IntellJ IDEA. Para rodar o projeto abra-o no IntellJ e o execute. Caso não possua siga estes passos.
      
      * -> O projeto neste repositorio já está pré-compilado, logo basta apenas entrar na pasta correta e executar os seguintes comandos:
      
      3.1 -> Após baixar o projeto navegue na linha de comando para a seguinte diretorio:
      
            <caminho_onde_salvou_o_projeto>/GympassTest/out/production/GympassTest
      
      3.2 -> Para "rodar" o projeto digite o seguinte comando e após isto tecle enter:
      
            java corrida.log.reader.Main <caminho_log_corrida_input> <caminho_relatorio_de_saida>
            
            OBS: corrida.log.reader.Main é o package da classe principal. <caminho_log_corrida_input> e <caminho_relatorio_de_saida> são
            respectivamente o caminho onde se encontra o arquivo de log da corrida e o caminho onde será salvo o relatório final da
            corrida.
       
      3.3 -> Após "rodar" este comando a solução irá realizar os cálculos necessários e gerar o relatorio
